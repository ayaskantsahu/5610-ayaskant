About the experiment

effect() method applies an animation effect to the elements without having to show or hide it. Animation can be many types. For example explode, bounce, Clip,etc


Code Snippets

Exploding effect
            $('#explodingBox1').click(function() {
               $( "#explodingBox1" ).effect({
                  effect: "explode",
                  easing: "easeInExpo",
                  pieces: 4,
                  duration: 3000
               });
            });
Shake Effect
Shake effect is shown with a vigourous moving effect which slowly settles down. 
            $('#explodingBox2').click(function () {
                $("#explodingBox2").effect("shake", {
                    times: 10,
                    distance: 100
                }, 3000, function () {
                    $(this).css("background", "#cccccc");
                });
            });
Blind Effect
Blind is used to generally hide or show elements with a gradual effect like blinds in the room. 
            $('#explodingBox3').click(function () {
                $( "#explodingBox3" ).hide( "blind", {times: 10, distance: 100}, 1000, callback );
            });
            function callback() {
                setTimeout(function() {
                    $( "#explodingBox3" ).removeAttr( "style" ).hide().fadeIn();
                }, 1000 );
            };
