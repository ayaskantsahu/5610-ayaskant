About the experiment

Slideshow are used as UX components on many homepages these days. They provide a very attractive and userfriendly way of browsing all the features of a website as pictures or as collection of latest news. This slider was built using raw jQuery without any plugin.


Code Snippets

This javascript code snippet enables the slideshow without any jquery-ui plugin.
          $(document).ready(function () {
            var Slideshow = {
                navigate: function () {
                    $('a', '#slideshow-nav').click(function (e) {
                        e.preventDefault();
                        var $a = $(this);
                        var slide = $($a.attr('href'));
                        var wrapper = $('#slideshow-wrapper');
                        wrapper.animate({
                            left: -slide.position().left
                        }, 1000, 'linear', function () {
                            $a.addClass('active').siblings().removeClass('active');
                        });
                    });
                },
                init: function () {
                    this.navigate();
                }
            };
            Slideshow.init();
        });
