About the experiment

Sometimes routing is preferred to be done serverside instead of client side. NodeJS allows easy routing by creation of paths with methods associated with it.


Code Snippets

These lines of code configure various routes based on the methods on the server. 
 
        app.get("/applications", function (req, res) {
            res.json(applications);
        })
        app.get("/addAppPage", function (req, res) {
            res.sendfile('./public/views/AddApp.html');
        })
        app.get("/loginPage", function (req, res) {
            res.sendfile('./public/views/login.html');
        })


These lines render the partial html onto the UI 
        function showAddApp()
        {
            $.ajax({
                url: "addAppPage",
                success: function (result) {
                    $("#view").html(result);
                }
            });
        }
        function showLogin() {
            $.ajax({
                url: "loginPage",
                success: function (result) {
                    $("#view").html(result);
                }
            });
        }
