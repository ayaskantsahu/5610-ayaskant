About the experiment

This is an extension of the previous week's experiment where new applications can be added and viewed as json. This time this addition and retrieval is done from ther NodeJS server.


Code Snippets

This snippet allows adding new application to new list and then returning the new list of applications 
 
        app.post("/saveApp", function (req, res) {
            var obj = req.body;
            applications.push(obj);
            res.json(applications);
        });
        app.get("/getApps", function (req, res) {
            res.json(applications);
        });
