Server side templating using Jade

Link to experiement


About the experiment

Every enterprise application might need server side templating as they may not want too much data to hand around on the client side. This can also affect the responsiveness of the UI as well. Jade is one such templating engine available on Node.js.


Code Snippets

These lines in server.js allow configuration of Jade and rendering of a jade file with data.
        app.set('views', __dirname);
        app.set('view engine', 'jade');

        app.post("/welcome", function (req, res) {
            var obj = req.body;
            console.log(obj);
            res.render('hello', { user : obj.user});
        });


This is the Jade template which we use to display the entered name.
        doctype html
            html(lang="en")
              head
                title= pageTitle
                script(type='text/javascript').
                  if (foo) {
                     bar(1 + 5)
                  }
              body
                h1 Jade - node template engine
                #container.col
                  p welcome #{user}
