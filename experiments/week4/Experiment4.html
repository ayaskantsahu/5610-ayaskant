About the experiment

AngularJS forms and controls can provide validation services, and notify users of invalid input. In this case The AngularJS directive ng-model binds the input elements to the model. The model object has two properties: user and email. Because of ng-show, the spans with color:red are displayed only when user or email is $dirty and $invalid.


Code Snippets

This directive decides to show the error container only when the input is invalid. 
 
        ng-show="myForm.email.$dirty && myForm.email.$invalid"
