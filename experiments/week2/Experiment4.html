About the experiment

The table has been made responsive by using media queries. These help us set us breakpoints at specific measurements in pixels based on which specific CSS class would be applied. We have used two breakpoints. One for normal screen and another for smaller screens.


Code Snippets

@media only screen and (min-device-width : 320px) and (max-device-width : 480px) { ... }
@media only screen and (max-width: 760px), (min-device-width: 768px) and (max-device-width: 1024px) { ... }
