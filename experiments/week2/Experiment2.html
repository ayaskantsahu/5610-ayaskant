About the experiment

Various 2D and 3D transforms are possible with CSS 3. Rotation is demonstration here
  • 2D rotation can be done with rotate() function
  • 3D rotation can be done with either rotateX(), rotateY() or rotateZ() methods
The examples show 2D and 3D rotation around X and Y axis only.


Code Snippets

x.style.transform = "rotate(" + n + "deg)"
y.style.transform = "rotateY(" + ny + "deg)"
