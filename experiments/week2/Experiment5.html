The below div is resizable



About the experiment

CSS 3 allows us to resize background images along with changing container size. This can be achieved with the background-size property. CSS 3 also allows us to make any container resizable by the user. A small triangle is shown at the lower right corner of the div in this case which can be dragged.


Code Snippets

background-size: 100% 100%;
resize: both;
