About the experiment

This demonstration consists of 3 different CSS 3 features, mostly relevant to borders.
  • Border Image : This will alow an image to be used as the border of an html element which was very difficult in previous CSS versions.
  • Border Radius : This feature enables rounded corners in borders. Earlier this was achieved by positioning images properly, which was difficult.
  • Box Shadow : This feature will render a shadow natively on any html element.


Code Snippets

border-image: url(/experiments/images/border.png) 20 stretch;
border-radius: 10px;
box-shadow: #808080 5px 5px;
