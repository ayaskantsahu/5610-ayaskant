About the experiment

Search feature can be easily implemented using mongoose and mongoDB as it internally allows usage of regular expressions in its APIs for pattern matching.


Code Snippets

This snippet allows searching users based on partial username
        app.get('/search/:term', function (req, res) {
            var input = req.params.term;
            User.find({ username: new RegExp(input, "i") }, function (err, data) {
                res.json(data);
            });
        });
