About the experiment

This experiment shows how passport.js can be used to log the user out. The existing session will be invalidated in this case. This is a demonstration of Passport.js with mongoDB and node.js Passport.js is the leading library for Node.js for authentication and authorization. It normally registers users into MongoDB and then authenticates based on it.


Code Snippets

These lines are used to Log a user out when he clicks on the log out button. The logout() method in request object basically invalidates the session of the user. 
 
        app.get('/logout', function(req, res) {
		    req.logout();
		    res.redirect('/');
	    });
