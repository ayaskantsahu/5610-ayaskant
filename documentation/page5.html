Assign Role

Assign Role allows the admin to assign roles to staff on a particular shift. This can be achieved by the following steps.
  1. Select a date to see shifts
  2. Press get shifts button
  3. Select a shift from the listed shifts
  4. Only the staff available in that shift are shown in the dropdown.
  5. Save the selections after deciding roles.
  6. Data is saved in MongoDB
