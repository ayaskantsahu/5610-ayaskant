Search Staff

This view allows the admin to search for a specific staff using auto suggest feature.
  1. Start typing a name. eg : ayas
  2. Select from the list of suggested names
  3. The results are fetched from whenIWork
  4. On click of 'show' statistics button, data is fetched from mongo and user's utilization is shown




JSON data

The data is heavily preprocessed on the UI side after getting the data from MongoDB. This new structure allows us to show the data progress bars with angular-bootstrap plugin..
