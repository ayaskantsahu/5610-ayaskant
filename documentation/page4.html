Dashboard

The dashboard shows the statistics about the current shift in progress. It fetches data from WhenIWork API after intelligently figuring out which shift is underway.

Steps :

  • Browser figures out current shift.
  • Browser calls express route.
  • nodeJS calls WhenIWork API
  • Data is visualized on UI using AgularJS
