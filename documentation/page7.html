All Staff Stats

This view allows the admin to view the percentage of allocation for each staff who has been assigned work.
  1. Select a date range to see shifts
  2. Press get shiftss button
  3. The results are fetched from mongoDB
  4. Results are preprocessed to bring them to UI usable structure
  5. Results are shown
