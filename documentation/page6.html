Time Range Stats

This view allows the admin to view all the assigned roles per shift in a certain time period.
  1. Select a date range to see shifts
  2. Press get assignments button
  3. The results are fetched from mongoDB
  4. Results are preprocessed to bring them to UI usable structure
  5. Results are shown
