Requirements

The requirements can be categorized into software and hardware requirements.

Hardware

  • Pentium class CPU with at least 1Ghz processor
  • 2 GB ram
  • Consistent internet connection with at least 512 Kbps speed for smooth experience.

Software

  • HTML 5 capable browser like google chrome, mozilla firefox or internet explorer 9
  • Node js v0.10
  • MongoDB v2.6.5
  • Windows/Mac/Linux OS
