Project based experiments - part 2

After setting up the basic infrastructure, I made some progress in building the features. I started making basic UI on top of services created previous week. Though these features are not polished yet, they are minimally functional and usable at this point of time.

  1. Added feature of assigning roles to staff on a specific shift. The data was fetched from 3rd party API call.
  2. The feature of dispaying these data from 3rd party API to help user assign a role to a specific staff was implemented.
  3. The assigned role is then saved into MongoDB for further analysis. Mongoose ODM was used for the same.
  4. Feature of searching user based on their partial names was implemented. This bootstrap-anguar plugin was very easy to use, thanks to the documentation.
  5. Previously built authentication was integrated into the project. Log out feature is still pending.
